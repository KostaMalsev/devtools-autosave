Privacy Policy for DevTools Autosave

Last Updated: February 24, 2025

Overview

DevTools Autosave is committed to protecting your privacy. This Privacy Policy explains how our extension handles data.

Data Collection and Usage

DevTools Autosave operates primarily locally within your browser. We:

  • DO NOT collect any personal information
  • DO NOT track your browsing activity
  • DO NOT send data to external servers (except when explicitly using the GitHub Gist feature)
  • DO NOT share any information with third parties beyond GitHub (when using the Gist feature)

Local Storage Usage

The extension uses Chrome's local storage to:

  • Save your DevTools modifications between page reloads
  • Persist changes between browser sessions
  • Store only the DevTools edits you make

GitHub Integration (Optional)

The extension offers an optional feature to save your DevTools modifications to GitHub Gists:

  • This feature is completely optional and must be explicitly activated by you
  • When you choose to save to GitHub, you will need to authenticate with your GitHub account
  • We use GitHub's OAuth2 authentication flow to securely connect to your account
  • Your GitHub credentials are never stored by our extension
  • We only receive and use an authentication token to create Gists on your behalf
  • The content saved to GitHub Gists contains only your DevTools modifications
  • You control whether Gists are created as public or private

Data Control

You maintain full control over your data:

  • All data is stored exclusively in your browser's local storage
  • Data persists until you clear your browser data
  • You can remove stored data at any time through your browser settings
  • For GitHub Gists, you can manage or delete them through your GitHub account
  • You can revoke the extension's access to your GitHub account at any time through GitHub settings

Required Permissions

The extension requires the following permissions:

  • Storage: To save your DevTools modifications locally
  • DevTools: To monitor and preserve your DevTools changes
  • Identity: Only used for GitHub authentication when you choose to save to Gists
  • Tabs: To associate modifications with their respective web pages

Updates to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.

Contact

If you have any questions about this Privacy Policy, please open an issue on our GitHub repository.
